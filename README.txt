Policy Validator 3.0

The Solution contains two projects - PolicyValidator and PolicyValidatorSetup. The PolicyValidatorSetup project depends on the PolicyValidator project and is used to create the MSI.

To build a new version, make the necessary code changes, right click on the PolicyValidatorSetup and click Rebuild. The new MSI can be found under policyValidatorSetup\Debug or policyValidatorSetup\Release folder based on the build mode.
